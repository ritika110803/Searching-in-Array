This project demonstrates the implementation of the Linear Search algorithm using the C programming language.
Linear Search is a simple searching algorithm that checks each element of the array one by one until the desired element is found or the array ends.
 Program Description
The program contains a predefined array:
{10, 25, 30, 45, 50}
It asks the user to enter a number to search.
It searches the element using a for loop.
Displays:
✅ The position of the element (if found)
❌ "Element not found" (if not found)
How the Algorithm Works
Start from the first element (index 0).
Compare each element with the entered number.
If a match is found:
Display its position.
Stop the loop.
If no match is found after checking all elements:
Display "Element not found".
| Case       | Complexity |
| ---------- | ---------- |
| Best Case  | O(1)       |
| Average    | O(n)       |
| Worst Case | O(n)       |
